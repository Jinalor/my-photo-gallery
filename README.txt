# 写真ギャラリー テンプレート

このテンプレートはHTMLとTailwind CSSを使った静的な写真ギャラリーです。
- `index.html`：トップページ
- `style/output.css`：Tailwind CDN経由のスタイル
- `images/`：ここに表示したい写真を入れてください（例：photo1.jpg など）

## 公開方法（GitHub Pages）
1. GitHubに新しいリポジトリを作成
2. このフォルダの内容をアップロード
3. リポジトリのSettings → Pages → Branch: main/root に設定
4. 数分後に `https://あなたのユーザー名.github.io/リポジトリ名/` で公開されます
